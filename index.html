SZA Text to Speech

Privacy Policy

Last updated: March 12, 2026

SZA Text to Speech is a browser extension that reads highlighted text aloud using voices installed inside the extension. This Privacy Policy explains what data the extension handles, how it is used, and what is stored.

What the Extension Does

The extension has a single purpose: to let a user highlight text on a webpage and hear that text spoken aloud using a selected installed voice.

Data the Extension Handles

  • Highlighted webpage text that the user explicitly selects to be read aloud.
  • User settings such as selected voice, speaker, speed, volume, and interface preferences.
  • Voice files that the user chooses to install inside the extension.

How Data Is Used

  • Highlighted text is used only to generate speech for the user's request.
  • Settings are used only to remember the user's preferred voice and playback options.
  • Installed voice files are used only to make speech playback available inside the extension.

What Is Stored

  • Voice, speaker, speed, volume, and interface settings are stored using browser extension storage.
  • Installed voice files are stored locally inside the extension using the browser's local storage mechanisms.

What Is Not Collected

  • No account information is collected.
  • No passwords, payment information, health information, or personal communications are collected.
  • No analytics or advertising trackers are used by the extension.
  • No browsing history is collected beyond the text the user explicitly chooses to read aloud.

Remote Requests

The extension may make network requests only when a user chooses to browse, preview, or install voices.

  • rhasspy.github.io is used to load the public voice catalog and sample previews.
  • huggingface.co is used to download voice files selected by the user.

Highlighted webpage text selected for speech is processed within the extension and is not sent to the extension publisher's own server.

Data Sharing

The extension does not sell user data. The extension does not transfer user data to third parties for advertising, profiling, creditworthiness, or other unrelated purposes.

How Long Data Is Kept

Stored settings and installed voices remain in the extension until the user changes them, deletes them, clears extension data, or uninstalls the extension.

User Control

  • Users can remove installed voices from the extension library at any time.
  • Users can change saved settings at any time.
  • Users can remove all extension data by uninstalling the extension or clearing its stored data in the browser.

Contact

For privacy questions, use the support contact listed on the extension's store page or project page.
